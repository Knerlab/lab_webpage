Privacy Notice

This website records basic visit analytics for operational and research-outreach metrics.

What We Collect

  • Visit timestamp, page path, response status.
  • Masked IP address (example: 192.168.1.*).
  • Approximate location fields from proxy/CDN headers when available (country/city).
  • Browser user-agent and referer URL (truncated for storage limits).

How We Protect Data

  • Raw IP is not stored in analytics files; only masked IP is saved.
  • An anonymized visitor ID is generated using one-way hashing.
  • Analytics data is stored locally as compact monthly CSV files.

Storage

Analytics files are stored under analytics_data/visits_YYYYMM.csv.
